TableSchema

Amazon DynamoDB
Available in AWSPowerShell, AWSPowerShell.NetCore and AWS.Tools.DynamoDBv2

Description

An object of this type is output from the New-DDBTableSchema cmdlet and updated using the Add-DDBKeySchema and Add-DDBIndexSchema cmdlets. It is used to easily construct and supply key and index schema to the New-DDBTable cmdlet for creation of an Amazon DynamoDB table.

This type is not intended to be used directly.
